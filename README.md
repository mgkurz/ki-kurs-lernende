# Künstliche Intelligenz verstehen — Grundlagenkurs (OER)

Interaktiver Grundlagenkurs zu Künstlicher Intelligenz in vier Teilen, in einem Rutsch
durcharbeitbar. Der ganze Kurs ist eine einzige HTML-Datei ohne externe Abhängigkeiten
(keine CDNs, Fonts, Tracker), datenschutzkonform. Am Ende: ein gemeinsamer Lernzettel
und eine Teilnahmebescheinigung. Zielgruppe: Sekundarstufe.

**Live:** <https://mgkurz.github.io/ki-kurs-lernende/>

## Bearbeiten und veröffentlichen

Der komplette Kurs steckt in `index.html`. Lokal ansehen: Datei im Browser öffnen.
Ändern, dann `git push` — GitHub Pages veröffentlicht automatisch (Wurzel von `main`).
